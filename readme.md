# Api rest example

## api rest with 3 models, clients products and categories using mongodb with mongoose.


## Initial configuration

1 ) Clone repository:

```bash
git clone https://github.com/pabloroque1/api-rest.git
```

2 ) Initial installation of proyect:

```bash
cd api-rest
```

```bash
npm install
```

3 ) Run proyect:

```bash
npm start
```
